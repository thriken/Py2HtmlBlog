## About This Blog

This is a personal blog powered by **Py2HtmlBlog** — a Python static blog generator.

### How It Works

1. Write posts in Markdown with frontmatter metadata
2. Run `python main.py publish`
3. Static HTML is generated under `wwwroot/`
4. Deploy via Git, FTP, or SFTP -- no database needed

### Features

- Markdown writing with YAML frontmatter
- Categories and tags
- RSS feed and XML sitemap
- 19 built-in themes (including iNove from WordPress)
- ZIP template packages for easy sharing
- PyQt5 desktop GUI editor
- FTP / SFTP / Git deployment
- URL migration compatible with ThingamaBlog

# Hello World — 欢迎使用 Py2HtmlBlog

## 快速开始

Py2HtmlBlog 是一个用 Python 编写的静态博客生成器。这篇文章将帮助你完成第一次发布。

### 1. 复制示例配置

```bash
cp -r example-data/* data/
```

### 2. 编辑站点配置

编辑 `data/info.json`，修改以下字段：

- **site_name** — 你的博客名称
- **site_url** — 你的博客网址
- **author** — 你的名字
- **template** — 选择主题（default / inove 等）

### 3. 选择主题

Py2HtmlBlog 内置了 **19 套主题**，包括：

| 主题 | 来源 |
|------|------|
| default | 原创，简洁现代风格 |
| inove | WordPress iNove 1.2.3 移植 |
| clean / georgia_blue / 等 | ThingamaBlog 原装模板 |

在 `info.json` 中设置 `"template": "default"` 即可切换。

### 4. 写文章

在 `data/post/` 目录下创建 Markdown 文件，命名格式为 `YYYYMMDD_N.md`：

```markdown
---
id: 1
title: 文章标题
datetime: 2026-07-22 12:00:00
slug: my-post
category: tech
tags: python, blog
status: published
---

你的文章内容...
```

### 5. 生成并发布

```bash
# 生成静态站点
python main.py publish

# 预览（wwwroot/ 目录）
# 部署到 GitHub Pages
python main.py deploy
```

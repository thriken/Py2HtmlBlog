# Markdown 写作指南

Py2HtmlBlog 使用 Markdown 作为写作格式。这篇文章介绍支持的语法。

## 标题

使用 `#` 标记标题级别，从一级到六级：

## 文字样式

- **粗体**：`**粗体**`
- *斜体*：`*斜体*`
- `行内代码`：`` `行内代码` ``

## 链接与图片

- 链接：`[文字](https://example.com)`
- 图片：`![替代文字](/path/to/image.png)`

## 代码块

使用三个反引号包裹代码块，可以指定语言：

```python
def hello():
    print("Hello, Py2HtmlBlog!")
```

## 列表

无序列表：

- 项目一
- 项目二
  - 嵌套项目
- 项目三

有序列表：

1. 第一步
2. 第二步
3. 第三步

## Frontmatter 字段说明

每篇文章顶部的 `---` 之间是 YAML 格式的元数据：

| 字段 | 必填 | 说明 |
|------|------|------|
| id | 是 | 文章唯一编号 |
| title | 是 | 文章标题 |
| datetime | 是 | 发布时间（YYYY-MM-DD HH:MM:SS） |
| slug | 是 | URL 友好别名 |
| category | 否 | 分类 slug |
| tags | 否 | 标签，逗号分隔 |
| status | 是 | `published` 或 `draft` |

> **提示**：使用 `python main.py new "文章标题"` 可以快速创建新文章的模板文件。
